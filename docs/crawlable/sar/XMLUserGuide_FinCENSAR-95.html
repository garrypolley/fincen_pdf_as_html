Element Restriction(s):
Record this element for the Financial institution where the account is held (41) party type when the
associated account is closed; otherwise, do not record this element.
Value Restriction(s):
The value provided must adhere to the following requirements: "Y" (to indicate "Account closed")
Remarks:
The batch submission will be rejected if any of the following conditions are true:
The element contains a value not equal to "Y" or null.
Example:
<fc2:AccountClosedIndicator>Y</fc2:AccountClosedIndicator>
